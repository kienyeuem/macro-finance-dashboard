# macro-finance-dashboard
Một ứng dụng web dashboard tương tác được xây dựng bằng Python, giúp theo dõi và phân tích các chỉ số kinh tế vĩ mô quan trọng từ dữ liệu thực tế của Ngân hàng Thế giới (World Bank).  Dự án này thể hiện khả năng làm việc với API, xử lý dữ liệu (Data Cleaning) và trực quan hóa dữ liệu (Data Visualization).
